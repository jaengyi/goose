# 레시피

goose 레시피는 goose가 특정 작업을 수행하는 데 필요한 모든 세부 사항이 포함된 파일입니다. 하나의 파일에 담겨 있기 때문에 git과 같은 버전 관리 시스템을 포함하여 파일을 공유하는 모든 일반적인 방법으로 쉽게 공유할 수 있습니다. 가장 간단한 레시피부터 시작해 봅시다.

## 가장 간단한 레시피

가장 간단한 레시피는 기본적으로 프롬프트일 뿐입니다. Slack이나 이메일로 프롬프트를 공유할 수 있으니 그다지 유용하지 않아 보일 수 있지만, 사용자가 에이전트에게 원하는 것을 얻지 못하는 가장 중요한 이유는 프롬프트가 너무 짧고 프롬프트를 충분히 반복하지 않기 때문입니다. 프롬프트를 텍스트 파일에 보관하면 이 두 가지 문제에 모두 도움이 됩니다.

다음은 유럽 여행을 계획하는 레시피입니다:

```yaml
title: Trip planner
description: Plan your next trip
prompt: |
 Help the user plan a trip to Europe for 14 days.
 Create a detailed itinerary that includes:
  - places to visit
  - activities to do
  - local cuisine to try
  - a rough budget estimate
```

명령줄에서 다음과 같이 실행할 수 있습니다:

```sh
goose run --recipe trip.yaml
```

## 확장 프로그램

goose 레시피에는 실행 중 goose가 사용할 수 있는 [확장 프로그램](/docs/guides/recipes/recipe-reference#extensions)을 지정할 수 있는 섹션이 있습니다. goose는 지정한 것만 사용합니다.

유럽 여행 중에 좋은 날씨를 보장하고 싶다고 가정해 봅시다. 레시피에 날씨 확장 프로그램(이 예제에서는 MIT 라이선스 하에 TuanKiri의 [weather-mcp-server](https://github.com/TuanKiri/weather-mcp-server)를 사용)을 추가하고, 프롬프트를 약간 수정하면 goose가 여행에 도시를 추가하기 전에 날씨를 확인합니다.

```yaml
title: Trip planner
description: Plan your next trip
prompt: |
 Help the user plan a trip to Europe for 14 days. Create a detailed itinerary that includes:
  - places to visit
  - activities to do
  - local cuisine to try
  - a rough budget estimate
 Ensure that the user has good weather throughout their trip. Optimize their trip based on the forecast in potential locations.
extensions:
  - type: stdio
    name: weathermcpserver
    cmd: /Users/svega/Development/weather-mcp-server/weather-mcp-server
    args: []
    timeout: 300
    description: "Weather data for trip planning"
    env_keys:
      - WEATHER_API_KEY
```

## 매개변수

매개변수를 추가하여 레시피를 동적으로 만들 수 있습니다. 매개변수는 레시피 사용자가 제공하는 변수입니다. 각각 데이터 유형과 필수, 선택 또는 사용자 제공 여부를 정의하는 요구 사항 필드가 있습니다. 목적지와 여행 기간에 대한 매개변수를 추가하여 여행 레시피를 일반화할 수 있습니다:

```yaml
parameters:
  - key: destination
    input_type: string
    requirement: required
    description: Destination for the trip. Should be a large region with multiple climates.
  - key: duration
    input_type: number
    requirement: required
    description: Number of days for the trip.
```

레시피는 `{{ destination }}`과 같은 변수를 삽입할 수 있는 템플릿 시스템을 사용하며, 제공하는 실제 값으로 채워집니다. 올바른 세부 정보로 프롬프트를 업데이트한 후, 아프리카 14일 여행 계획을 얻기 위해 다음과 같이 새 레시피를 실행할 수 있습니다:

```sh
goose run --recipe trip.yaml --params destination=Africa --params duration=14
```


## 설정

기본적으로 goose는 이미 선택한 `temperature`와 `model`을 사용하며, 이것이 대개 잘 작동합니다. 하지만 때로는 더 많은 제어가 필요할 수 있습니다. 예를 들어, 여행 계획과 같은 주관적인 작업을 수행할 때 `temperature` 설정을 높이면 도움이 될 수 있습니다. temperature를 창의성 다이얼처럼 생각하세요 - 높을수록 결과가 더 다양하고 예상치 못합니다. 첫 번째 제안이 정확히 맞지 않으면 사용자는 레시피를 다시 실행하여 새로운 것을 얻을 수 있습니다.

특정 레시피에 사용할 AI 프로바이더와 모델을 지정할 수도 있습니다:

```yaml
settings:
  goose_provider: "anthropic"
  goose_model: "claude-sonnet-4-20250514"
  temperature: 0.8
```

사용 가능한 설정:
- `goose_provider`: AI 프로바이더 (예: "anthropic", "openai")
- `goose_model`: 특정 모델 이름
- `temperature`: 창의성/무작위성 제어 (0.0-1.0, 높을수록 더 창의적)

이 설정은 이 레시피가 실행될 때 기본 goose 구성을 재정의합니다.

## 외부 파일

때로는 프롬프트에 모든 데이터를 넣지 않고 에이전트에게 추가 정보에 대한 액세스를 제공하고 싶을 것입니다. 모든 것을 붙여넣는 대신 데이터를 별도의 파일에 보관하고 레시피가 그것을 가리키도록 할 수 있습니다.

이를 돕기 위해 레시피에는 `{{ recipe_dir }}`이라는 내장 변수가 포함되어 있어 레시피와 함께 저장된 파일을 참조할 수 있습니다. 예를 들어, [Kaggle](https://www.kaggle.com/datasets/ramjasmaurya/unesco-heritage-sites2021?resource=download)에서 UNESCO 목록을 다운로드하여 여행 계획 레시피에서 사용할 수 있습니다.

그런 다음 프롬프트에서 파일을 다음과 같이 참조합니다:

```yaml
prompt: |
 You can use the \{\{ recipe_dir \}\}/unesco.csv file to
 check information on UNESCO world heritage sites to
 include in your travel plan.
```

파일을 읽기 위한 확장 프로그램도 지정해야 합니다:

```yaml
extensions:
 - type: builtin
   name: developer
   display_name: Developer
   timeout: 300
   bundled: true
```

여기서 관련 정보를 읽을 수 있는 기능을 제공하는 [Developer 확장 프로그램](/docs/mcp/developer-mcp)을 추가합니다.

# 10일 유럽 모험 일정

이 일정은 유럽에서 가장 아름답고 문화적으로 풍부한 세 나라인 프랑스, 이탈리아, 체코를 여행합니다. 세계적 수준의 박물관, UNESCO 세계유산, 맛있는 요리, 활기찬 현지 문화를 경험하게 됩니다.

#### 1-3일차: 파리, 프랑스 🇫🇷

**1일차: 파리 도착**
- **오전**: 샤를 드골 공항 도착, 호텔로 이동
- **오후**: 센 강을 따라 여유롭게 산책, 노트르담 대성당 방문 (재건 중이므로 외부 관람)
- **저녁**: 라틴 지구에서 저녁 (예산: €30-40)
  - 클래식 프렌치 어니언 수프와 코코뱅 맛보기

**날씨 예보**: 27°C(81°F)의 쾌적한 기온, 부분적으로 흐림

**2일차: 파리 하이라이트**
- **오전**: 루브르 박물관 방문 (예산: €17)
- **오후**: 튈르리 정원과 샹젤리제 탐험
- **저녁**: 일몰을 위한 에펠탑 방문 (예산: 정상 접근 €26.80)
  - 트로카데로 근처에서 저녁 (예산: €35-45)
  - 에스카르고와 뵈프 부르기뇽 맛보기

**날씨 예보**: 31°C(88°F)로 따뜻함, 맑은 하늘

**3일차: 베르사유 당일 여행**
- **오전**: 베르사유 궁전 당일 여행, UNESCO 세계유산 (예산: 궁전 입장 €21)
- **오후**: 웅장한 정원 탐험
- **저녁**: 파리로 돌아와 몽마르트르에서 저녁 (예산: €30-40)
  - 크레페와 오리 콩피 맛보기

**날씨 예보**: 30°C(86°F)로 따뜻함, 약간의 비 가능성

#### 4-6일차: 로마, 이탈리아 🇮🇹

**4일차: 로마로 이동**
- **오전**: 파리에서 로마로 비행 (예산: €100-150)
- **오후**: 호텔 체크인, 스페인 계단과 트레비 분수 탐험
- **저녁**: 트라스테베레 지역에서 저녁 (예산: €25-35)
  - 정통 카치오 에 페페와 카르보나라 파스타 맛보기

**날씨 예보**: 35°C(95°F)로 더움, 맑은 하늘

**5일차: 고대 로마**
- **오전**: 콜로세움과 로마 포럼 방문 (예산: 통합 티켓 €16)
- **오후**: 팔라티노 언덕과 치르코 마시모
- **저녁**: 캄포 데 피오리 근처에서 저녁 (예산: €30-40)
  - 로마 스타일 피자와 살팀보카 알라 로마나 맛보기

**날씨 예보**: 35°C(95°F)로 더움, 대체로 맑음

**6일차: 바티칸 시국**
- **오전**: 바티칸 박물관과 시스티나 성당 (예산: €17)
- **오후**: 성 베드로 대성당과 광장 (UNESCO 세계유산)
- **저녁**: 프라티 지구에서 저녁 (예산: €30-40)
  - 수플리(라이스볼)와 오소부코 맛보기

**날씨 예보**: 34°C(93°F)로 더움, 부분적으로 흐림

#### 7-10일차: 프라하, 체코 🇨🇿

**7일차: 프라하로 이동**
- **오전**: 로마에서 프라하로 비행 (예산: €100-150)
- **오후**: 호텔 체크인, 구시가지 광장 탐험
- **저녁**: 구시가지에서 저녁 (예산: €20-30)
  - 전통 굴라시와 스비치코바(마리네이드 소고기) 맛보기

**날씨 예보**: 29°C(84°F)로 쾌적함, 부분적으로 흐림

**8일차: 프라하 성과 말라 스트라나**
- **오전**: 프라하 성 단지 방문 (UNESCO 세계유산) (예산: 250 CZK/€10)
- **오후**: 말라 스트라나와 카를교 탐험
- **저녁**: 블타바 강이 보이는 곳에서 저녁 (예산: €25-35)
  - 트르들니크(달콤한 페이스트리)와 만두를 곁들인 로스트 덕 맛보기

**날씨 예보**: 22°C(72°F)로 서늘함, 뇌우 가능성

**9일차: 유대인 지구와 문화 유적지**
- **오전**: 유대인 지구(요제포프)와 시나고그 방문
- **오후**: 시민회관과 화약탑
- **저녁**: 공연이 있는 전통 체코 민속 저녁 (예산: €35-45)
  - 체코 맥주와 콜라체 페이스트리 맛보기

**날씨 예보**: 24°C(75°F)로 쾌적함, 대체로 맑음

**10일차: 출발**
- **오전**: 프라하 부티크에서 마지막 쇼핑
- **오후**: 출발 비행을 위해 공항으로 이동

**날씨 예보**: 24°C(75°F)로 쾌적함, 대체로 맑음

#### 여행 팁
1. **날씨**: 예보에 따르면 모든 목적지에서 20-35°C(68-95°F)의 따뜻한 날씨에 대비하세요. 프라하의 서늘한 저녁을 위해 가벼운 재킷을 가져가세요.
2. **통화**: 프랑스와 이탈리아는 유로(€), 체코는 체코 코루나(CZK).
3. **교통**: 각 도시에서 지하철/대중교통 패스를 구매하여 비용 절감.
4. **예약**: 긴 줄을 피하기 위해 주요 명소를 미리 예약하세요, 특히 루브르, 바티칸 박물관, 에펠탑.
5. **물**: 특히 더운 로마에서 리필 가능한 물병을 가져가세요.
6. **언어**: 관광 지역에서는 영어가 널리 사용되지만 각 언어의 기본 표현을 배우세요.

이 일정은 세 가지 독특한 유럽 지역에서 역사, 문화, 요리의 완벽한 조화를 제공합니다. 날씨는 관광에 훌륭할 것이며, 대부분 맑고 따뜻한 날씨가 예상됩니다. 유럽 모험을 즐기세요!
