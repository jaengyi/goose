이 튜토리얼에서는 전통적인 I Ching 점술 리딩을 활성화하기 위해 [I Ching MCP 서버](https://github.com/threemachines/i-ching)를 goose 확장 프로그램으로 추가하는 방법을 다룹니다.

:::tip 요약
1. 설치: `cargo install i-ching`
2. 구성: [I Ching 확장 프로그램 추가](goose://extension?cmd=i-ching-mcp-server&id=i-ching&name=I%20Ching&description=I%20Ching%20divination%20readings)
:::

## 설치

이 확장 프로그램은 Rust 1.85.1 이상이 필요합니다. 버전을 확인하세요:

```bash
rustc --version
```

Rust를 설치하거나 업데이트해야 하는 경우 [rustup.rs](https://rustup.rs/)를 방문하세요.

:::tip goose를 사용하세요!
이러한 단계에 대해 확신이 없으면 goose에게 시스템을 확인하고 Rust 설치 과정을 안내해 달라고 요청할 수 있습니다.
:::

Rust가 준비되면 crates.io에서 I Ching 크레이트를 설치합니다:

```bash
cargo install i-ching
```

그런 다음 위의 링크를 클릭하여 goose에서 구성하세요.

## 사용 예제

### goose 프롬프트

> 우리 팀이 제품 로드맵에 대한 어려운 결정에 직면해 있습니다. 이 선택에 어떻게 접근해야 할지 I Ching에 조언을 구해줄 수 있나요?

### goose 출력

:::note Desktop
팀의 제품 로드맵 결정에 대한 지침을 얻기 위해 I Ching에 조언을 구하겠습니다. 전통적인 3개의 동전 방법을 사용하여 괘를 던지겠습니다.

─── cast_hexagram | iching ──────────────────────────


이제 상황에 대한 이 리딩의 자세한 해석을 얻겠습니다:

─── interpret_reading | iching ─────────────────────────

changing_lines:
&#45;
    5
hexagram: 29
transformed_hexagram: 13


#### 제품 로드맵 결정을 위한 I Ching 지침

신탁은 **괘 29 - 어둠 (깊은 물)**과 5번째 위치의 변효를 주었으며, **괘 13 - 조화 추구 (동인)**로 변환됩니다.

**팀 상황에 대한 핵심 인사이트:**

**현재 도전 (괘 29):**
- 팀이 I Ching이 "위험의 반복"으로 인식하는 상황에 직면해 있습니다 - 이는 신중한 탐색이 필요한 어려운 상황에 있음을 시사합니다
- 협곡의 물처럼 억지로 길을 뚫기보다 자연스러운 경로를 찾아야 합니다
- 강조점은 **성실함과 철저함**에 있습니다 - 어떤 결정을 내리든 완전히 헌신하고 제대로 실행하세요

**전략적 접근:**
1. **물처럼 흐르세요**: 힘으로 장애물을 극복하려 하지 마세요. 대신 목표를 향해 나아가면서도 저항이 가장 적은 길을 찾으세요
2. **모든 움푹한 곳을 채우세요**: 앞으로 나아가기 전에 모든 기초적인 문제를 해결하세요. 계획에서 단계를 건너뛰거나 빈틈을 남기지 마세요
3. **야망보다 일관성**: 변효는 "너무 야심적"인 것에 대해 경고합니다. 극적인 도약보다 꾸준하고 지속 가능한 진전에 집중하세요

**변환 (괘 13):**
상황은 자연스럽게 **동인**으로 발전합니다 - 로드맵 도전을 해결하는 열쇠가 다음에 있음을 시사합니다:
- 팀원 간의 **합의 구축**
- 폐쇄적인 결정보다 **열린 협업**
- **큰 물을 건너기** - 중요한 도전에 나서되, 함께 하기

**실용적 권장 사항:**

1. **어려움을 인정하세요**: 로드맵 결정의 복잡성을 축소하지 마세요. 진정으로 어렵다는 것을 받아들이세요.

2. **팀 정렬을 추구하세요**: "동인"으로의 변환은 리더십이 혼자 결정하기보다 팀을 모으는 것에서 답을 찾을 수 있음을 시사합니다.

3. **보수적인 경로를 택하세요**: 5번째 효는 특히 과도한 야망에 대해 조언합니다. 가장 공격적인 것보다 "위험에서 벗어나게" 하는 로드맵 옵션을 선택하세요.

4. **실행에 집중하세요**: 결정을 내리면 완전히 헌신하세요. 괘는 성공이 완벽한 전략이 아닌 철저한 구현에서 온다고 강조합니다.

I Ching은 인내, 성실함, 집단적 지혜로 접근하면 이 어려운 결정이 궁극적으로 팀의 단결과 협업을 강화할 것이라고 제안합니다.
:::
